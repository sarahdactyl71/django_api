# Contact API

### Requirements
1. Python Version

### Running the Server
1. clone this repository
2. navigate to the inner `django_api` folder, `cd django_api`
3. run `python manage.py runserver`

### Testing Endpoints
I used Postman to test any enpoints that updated information.
